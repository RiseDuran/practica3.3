Navegadores web

Un navegador web (en Inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, interpretando la información de distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualizacion de documentos de texto, posiblemente con recursos multimedia incrustados.

Ejemplo de navegadores web

Safari: Safari es un navegador web de código cerrado desarrollado por Apple Inc. Está disponible para macOS, IpadOS e iOS.

Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux, Android, iOS, y Microsoft WIndows coordinado por la corporacion Mozilla y la Fundacion Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque deriva de proyectos de codigo abierto.
